Busidex Android Mobile App

For those using the Busidex app for Android, you may not be aware that there is a brand new version available in the Google Play store. This new version is faster, easier to use and has more features than the old app. You can get the new app here: https://play.google.com/store/apps/details?id=com.busidex.

But I already have the App...

If you already have the app installed, be sure you have the current version (v1.2.3). It's possible that you may have an old version that isn't getting the updates. If your Busidex app has this icon...
Old Icon
... then you should uninstall it and get the new app. You'll know you have the new app if you see this icon after it's installed...
New Icon

System Requirements

Just a small note, the new Busidex app for Android requires Android version 4.1 or higher. If you are using a version older than that, please contact busidex.help@gmail.com and we will try to help you out. At last count, we are supporting 4,942 Android devices, so hopefully yours is one of them :)
